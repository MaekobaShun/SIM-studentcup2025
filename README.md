## 概要

本コンペティションでは、LLM（大規模言語モデル）によって生成された「架空の映画・アニメのあらすじ」から、元ネタとなっている実在の有名作品（映画・アニメ・漫画）のタイトルを予測します。

この文章には、複数の実在作品の「世界観」や「テーマ」が複雑に織り交ぜられており、参加者はこのテキストから元ネタとなった2作品を見抜くモデルを作成し、予測精度を競います。

## タスク

あらすじの元ネタになっている有名作品を推定する。
人間なら直感的にできるこの作業を、機械的にどう実現するか？という問いに挑むことで、自然言語の意味解析・情報抽出・テキスト分類など、AIの本質に迫る体験ができます。

## データ

- **既存作品**: 有名作品50作品のID、カテゴリ、タイトル、あらすじが日本語で公開されます（古典的映画から比較的最近のアニメーションまで幅広いジャンル）
- **架空作品**: 340作品が提供されます。それぞれは、既存作品から2作品を選び、世界観やテーマを合成したものです

## 提出形式

それぞれの架空作品について、元ネタになっている2作品を推定し、その作品IDのペアが書かれたCSVファイルを提出します。

## 評価方法

作品IDのペアが一致したかどうかの正解率（accuracy）で評価します。

- 素材の2作品 a, b の順序は問いません（例: "a=1, b=2" と "a=2, b=1" はどちらでも正解）
- 片方の一致のみの場合は正解となりません（例: 素材が1,2のとき、"a=1, b=3"は不正解）

## 例題

以下の架空のあらすじは、「スターウォーズ」と「鬼滅の刃」を混ぜたものです。

```
星々の空間を渡る旅で、辺境の集落に生きる若者は、恐怖統治への抵抗に身を投じる。
心を奪われた妹を人に戻す願いを胸に、冷たい金属の通路を走り、要塞からの脱出を図る。
物語の語り口は神話と西部劇の融合として進み、倒れても立ち上がる彼は、鍛錬と実戦の成長を重ねる。
やがて小型戦闘機の決戦に挑むが、復讐と救済の両立を求め、怒りに飲まれぬ優しさで引き金をためらう。
社会の安寧を追うため、敵の過去にも目を向け、自己犠牲の是非を問い直す。
幾度も傷つき倒れながらも呼吸を整え、憎しみの渦に沈まぬよう自らを律する。
最後に彼が選ぶのは、破壐ではなく心を取り戻す一射。凍てついた空に新しい朝が差すとき、妹の瞳に人の光が戻り、誰もが恐怖ではなく希望で結ばれる。
```

**作品要素の例：**
- スターウォーズ: 辺境の集落に生きる若者、神話と西部劇の融合、小型戦闘機の決戦
- 鬼滅の刃: 妹を人に戻す願い、鍛錬と実戦の成長、傷つき倒れながらも呼吸を整え
